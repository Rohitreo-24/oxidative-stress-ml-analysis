# 🧬 Oxidative Stress Prediction using Machine Learning

## 📌 Overview
This project analyzes transcriptomic gene expression data of *Megalurothrips usitatus* to predict oxidative stress using machine learning models.

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib

## 🧠 Models Used
- Linear Regression
- Random Forest
- XGBoost
- Stacking Ensemble

## 📊 Features
- Data preprocessing & cleaning
- Feature scaling
- Model comparison
- Cross-validation
- Feature importance analysis

## 📈 Results
- Ensemble models outperformed baseline models
- XGBoost and Random Forest showed best accuracy

[Mendeley Dataset Link](https://data.mendeley.com/datasets/fv3sshb8ph/1)
